# Minecraft Server Control - UI デザイン仕様

このドキュメントでは、実装されている画面のデザインとレイアウトを説明します。

## 🎨 デザインシステム

### カラーパレット
- **プライマリ**: Green (#10B981 - Tailwind green-500)
- **セカンダリ**: Blue (#3B82F6 - Tailwind blue-500)
- **エラー**: Red (#EF4444 - Tailwind red-500)
- **警告**: Yellow/Orange (#F59E0B - Tailwind yellow-500)
- **背景**: Gray (#F9FAFB - Tailwind gray-50)
- **テキスト**: Gray (#374151 - Tailwind gray-700)

### タイポグラフィ
- **見出し**: 2xl-3xl, font-bold
- **本文**: base, font-normal
- **小文字**: sm-xs, text-gray-600

## 🔘 ボタン状態

### サーバー起動ボタン
- **有効時**: 緑色 (#10B981)、ホバーで濃い緑 (#059669)
- **無効時**: グレー (#D1D5DB)、カーソルは not-allowed
- **有効条件**: サーバーが停止中のみ

### サーバー停止ボタン
- **有効時**: 赤色 (#EF4444)、ホバーで濃い赤 (#DC2626)
- **無効時**: グレー (#D1D5DB)、カーソルは not-allowed
- **有効条件**: サーバーが稼働中のみ

### LINEログインボタン
- **色**: 緑色 (#10B981) - LINEのブランドカラー
- **アイコン**: LINEロゴSVG
- **ホバー**: 濃い緑 (#059669)

### 招待コード生成ボタン
- **色**: 青色 (#3B82F6)
- **ホバー**: 濃い青 (#2563EB)

### コピーボタン
- **通常時**: グレー (#E5E7EB)
- **コピー後**: 緑色 (#10B981) + "コピー済み"表示

## 📱 レスポンシブデザイン

### モバイル (< 768px)
- カードの幅: 最大幅 (max-w-md)
- ボタン: フル幅
- パディング: 小さめ (p-4)

### タブレット/デスクトップ (≥ 768px)
- カードの幅: 最大幅 (max-w-4xl)
- ボタン: グリッドレイアウト (2列)
- パディング: 標準 (p-6)

---

## 🎭 アニメーション

### ローディング状態
```
  ⟳  (回転するスピナー)
```
- 使用箇所: ページ読み込み、ステータス更新中

### 点滅アニメーション
- 起動中/停止処理中のステータスドット
- Tailwind の `animate-pulse` を使用

### ボタンクリック
- `active:scale-95` でクリック時に少し縮小

---

## 🎨 Tailwind CSS クラス一覧

### レイアウト
- `flex min-h-screen flex-col items-center justify-center`
- `w-full max-w-md` (ログイン)
- `w-full max-w-4xl` (ダッシュボード)

### カード
- `bg-white rounded-lg shadow-md p-6`

### ボタン
- `py-3 px-4 rounded-md font-semibold transition-colors`
- `bg-green-500 hover:bg-green-600` (起動)
- `bg-red-500 hover:bg-red-600` (停止)
- `bg-blue-500 hover:bg-blue-600` (招待コード)

### ステータス
- `bg-green-50 border-green-500` (稼働中)
- `bg-gray-50 border-gray-500` (停止中)
- `bg-yellow-50 border-yellow-500` (起動中)
- `bg-orange-50 border-orange-500` (停止処理中)
